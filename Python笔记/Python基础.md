## 行和缩进和编码

- Python使用缩进表示代码块；
- 缩进的空格数是可变的，但是同一个代码块的语句必须包含相同的缩进空格数；
- Python 通常是一行写完一条语句，但如果语句很长，也可以使用反斜杠`\`来实现多行语句；
- 在 [], {}, 或 () 中的多行语句，不需要使用反斜杠`\`

- 默认情况下，Python3 源码文件以 **UTF-8**编码，所有字符串都是 unicode 字符串。当然也可以为源码文件指定不同的编码。

## 变量

### **标识符**

- 第一个字符必须以字母（a-z, A-Z）或 **下划线_** ；
- 标识符的其他的部分由字母、数字和下划线组成；
- 大小写敏感；
- 无长度限制，但一般简洁，不超过二十字符；
- 禁止使用保留关键字
- 合法标识符：

```python
age = 25                # 普通变量名，最常见
user_name = "Alice"     # 用下划线连接单词，清晰易读
_total = 100            # 下划线开头通常表示“内部使用”或“私有”
MAX_SIZE = 1024         # 全大写通常表示“常量”（固定不变的值）
calculate_area()        # 函数名，动词+名词
StudentInfo             # 类名，首字母大写（驼峰命名法）
__private_var           # 双下划线开头，有特殊含义
```

- **非法标识符**
  
```python
2nd_place = "silver"    # 错误：以数字开头
user-name = "Bob"       # 错误：包含连字符
class = "Math"          # 错误：使用关键字
$price = 9.99          # 错误：包含特殊字符
for = "loop"           # 错误：使用关键字
```

### 变量与内存

- Python是动态语言，利用赋值语句`=`可以将任意数据类型赋值给变量；
- 解释器中赋值语句执行过程：在内存中创建值，在内存中创建变量并将其指向所赋的值；
- 变量也可以赋值给另一个变量，其实质是将变量指向原变量所指的数据。

### 常量
    在Python中，通常用全部大写的变量名表示常量，但事实上仍是一个变量，没有任何保护机制。

## I/O

### **输出**：`print()`函数

**格式化输出**：

- `%s` 字符串，也可以输出`%d`，`%f`
- `%d` 含正负的十进制整数
- `%f` 浮点数

```python
# 我的年龄是18岁
    age=18
    print('我的年龄是%d岁' % age)
# 我的名字叫TOM
    name = 'TOM'
    print('我的名字叫%s' % name )
# 我的体重是75.5公斤
    weight= 75.5
    print('我的体重是%.2f公斤' % weight) # %f中间插入“.2”表示保留几位小数
# 我的学号是0001
    stu_id = 1
    print('我的学号是%04d') 
    # %d中间插入“04”表示输岀的整数显示位数,不足以0补全,超出当前位数则原样输出
# 我的名字叫TOM，我的年龄是18岁
    print('我的名字叫%s，我的年龄是%d岁' %(name , age ) )
# 我的名字叫TOM，我明年19岁
    print('我的名字叫%s，我的年龄是%d岁' %(name , age+1 ) )
```

**格式化字符串**：`f ' 表达式 { } { } '`

```python

```

**`formant()`函数**：`"{} {}".format("", "")`

```python
>>>"{} {}".format("hello", "world")    # 不设置指定位置，按默认顺序
'hello world'
 
>>> "{0} {1}".format("hello", "world")  # 设置指定位置
'hello world'
 
>>> "{1} {0} {1}".format("hello", "world")  # 设置指定位置
'world hello world'
```

**数字格式化**

|I|格式化|O|说明|
|---|---|---|---|
|3.1415926|{:.2f}|3.14|保留小数点后两位|
|3.1415926|{:+.2f}|+3.14|带符号保留小数点后两位|
|2.7|{:.0f}|3|取整|
|5|{:0>2d}|05|左补零，宽度为2|
|10|{:x<4d}|10xx|右补x，宽度为4|
|1000000|{:,}|1,000,000|逗号分割数字|
|0.25|{:.2%}|25.00%|百分比格式|
|1000000000|{:.2e}|1.00e+09|指数表示|
|**{:>10d}**|**{:<10d}**|**{:^10d}**|**右对齐/左对齐/中间对齐**|

## 运算符号

### 算数

- `+`:两个对象相加
- `-`:得到负数或是一个数减去另一个数
- `*`:两个数相乘或是返回一个被重复若干次的字符串
- `/`:除法计算结果是浮点数
- `//`:往小的方向取整数
- `%`:返回除法的余数
- `**`:返回x的y次幂

### 比较

### 赋值

- `+=`:c += a 等效于 c = c + a
- `-=`:c -= a 等效于 c = c - a
- `*=`:c *= a 等效于 c = c * a
- `/=`:c /= a 等效于 c = c / a
- `//=`:往小的方向取整数
- `%=`:c %= a 等效于 c = c % a
- `**=`:c **= a 等效于 c = c ** a
- `:=`:海象运算符，主要目的是在表达式中同时进行赋值和返回赋值的值
